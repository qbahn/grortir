Getting Started

Requirements

  • Python 3.3+

Installation

Grortir can be installed with pip:

$ pip install grortir

or directly from the source code:

$ git clone https://github.com/qbahn/grortir.git
$ cd grortir
$ python setup.py install

Basic Usage

After installation, the package can imported:

$ python
>>> import grortir
>>> grortir.__version__
